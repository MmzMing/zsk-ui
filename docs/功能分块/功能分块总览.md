## 四、核心功能详细说明

### 4.1 路由系统

**技术栈：** React Router DOM v7.12.0  
**文件位置：** `src/router/`

**特点：**
- 使用 BrowserRouter 进行路由管理
- 采用 Lazy Loading 进行代码分割
- 支持三种布局：BasicLayout（基础布局）、AdminLayout（后台布局）、BlankLayout（空白布局）
- 使用 Suspense + Loading 组件处理加载状态

### 4.2 状态管理

**技术栈：** Zustand v5.0.9  
**文件位置：** `src/store/`

**状态模块：**
- **appStore**: 应用全局配置（主题、布局、语言等）
- **userStore**: 用户信息、Token 等
- **resumeStore**: 简历编辑器状态

### 4.3 布局系统

**技术栈：** React + HeroUI + Framer Motion  
**文件位置：** `src/layouts/`

**BasicLayout（前台布局）：**
- 响应式 Header（支持滚动隐藏）
- 导航栏（支持水平/垂直/混合/双排等多种模式）
- 页脚（包含社交链接、支持信息等）
- 主题切换、语言切换
- 用户菜单（登录/个人中心/退出）
- 回到顶部按钮

**AdminLayout（后台布局）：**
- 多种布局模式（水平/垂直/混合/双排/Dock）
- 侧边栏菜单（支持收起/展开）
- 多标签页功能（支持右键菜单）
- 面包屑导航
- 全屏切换
- 系统设置面板

### 4.4 API 请求层

**技术栈：** Axios v1.13.2 + TypeScript  
**文件位置：** `src/api/`

**特点：**
- Axios 实例封装（请求/响应拦截器）
- 统一的错误处理
- Mock 数据支持
- Token 自动注入（从 Cookie 读取）
- 支持 Mock 兜底机制

**API 分类：**
- **admin/**: 后台管理相关 API（菜单、用户、文档、视频等）
- **auth/**: 认证相关 API（登录、注册、登出）
- **front/**: 前台展示相关 API（首页、搜索、文档、视频等）

### 4.5 富文本编辑器

**技术栈：** Tiptap v3.15.3 + Draft JS v0.11.7 + React Quill v2.0.0  
**文件位置：** `src/components/Editor/`、`src/pages/Resume/`、`src/pages/Admin/Document/Edit.tsx`

**支持的编辑器：**
1. **Tiptap（主要）**: 现代富文本编辑器
   - 支持图片、链接、表格
   - 可扩展架构
   - 类型安全

2. **Draft JS**: Facebook 开源编辑器
   - 插件化架构
   - 高度可定制

3. **React Quill**: Quill 的 React 封装
   - 简单易用
   - 功能完善

### 4.6 视频播放器

**技术栈：** Vidstack v1.12.13 + React Player v3.4.0  
**文件位置：** `src/components/VideoPlayer/`

**功能：**
- 支持播放/暂停/进度条
- 音量控制
- 全屏切换
- 移动端适配

### 4.7 动画与动效

**技术栈：** Framer Motion v12.24.10 + GSAP v3.14.2 + Lenis v1.3.17  
**文件位置：** `src/components/Motion/`

**动画组件：**
- PageTransitionWrapper: 页面过渡动画
- HsrIntroAnimation: 星穹铁道风格入场动画
- SmoothScroll: 平滑滚动
- AnimatedContent/AnimatedList: 内容动画
- ClickSpark: 点击火花效果
- StaggeredMenu: 错落菜单动画
- 等等...

### 4.8 3D 与粒子效果

**技术栈：** Three.js v0.182.0 + React Three Fiber v9.5.0 + React Three Drei v10.7.7  
**文件位置：** `src/components/Three/`、`src/components/ui/particles.tsx`

**组件：**
- ParticleBanner: 粒子背景
- Particles: 粒子效果组件（首页背景）

### 4.9 简历编辑器

**技术栈：** React + Tiptap + Zustand + HeroUI  
**文件位置：** `src/pages/Resume/`

**功能：**
- 基本信息编辑
- 多个模块管理（经历、技能等）
- 实时预览
- 工具栏

### 4.10 菜单管理

**技术栈：** React + rc-tree v5.13.1 + HeroUI  
**文件位置：** `src/pages/Admin/Personnel/Menu.tsx`

**功能：**
- 树形菜单展示
- 拖拽调整层级和排序
- 菜单编辑（名称、路径、图标、权限等）
- 批量删除
- 保存配置
