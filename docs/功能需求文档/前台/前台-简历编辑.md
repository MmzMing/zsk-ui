# 前台-简历编辑

**文件位置：** `src/pages/Resume/index.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 简历编辑
- **功能描述：** 提供在线简历编辑功能，支持用户创建和编辑个人简历
- **核心组件：**
  - `ResumeToolbar` - 顶部工具栏
  - `ResumeEditor` - 左侧编辑区
  - `ResumePreview` - 右侧预览区
- **交互设计：**
  - 支持编辑和预览两种模式切换
  - 桌面端：左右分栏布局（左侧编辑、右侧预览）
  - 移动端：标签页切换模式
  - 打印时隐藏编辑区，只显示预览内容

### 2. 编辑功能
- **功能描述：** 提供丰富的简历编辑功能
- **核心功能：**
  - 基本信息编辑
  - 多个模块管理（工作经历、教育经历、技能等）
  - 富文本编辑（使用 Tiptap 编辑器）
  - 实时预览编辑效果
- **工具栏：** 提供保存、导出、主题切换等功能

### 3. 数据管理
- **功能描述：** 使用 Zustand 管理简历编辑状态
- **状态管理：**
  - 简历模块数据
  - 编辑历史
  - 主题设置
- **数据获取：** 页面加载时自动获取简历数据

### 4. 响应式布局
- **功能描述：** 适配不同屏幕尺寸
- **布局适配：**
  - 桌面端：左右分栏
  - 平板端：响应式调整
  - 移动端：标签页切换
  - 打印模式：专门的打印样式
