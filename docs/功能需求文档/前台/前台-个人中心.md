# 前台-个人中心

**最后更新日期：** 2026-02-16

## 功能需求

### 1. 个人信息管理
- **功能描述：** 管理用户的个人信息
- **功能包括：**
  - 编辑个人资料（头像、昵称、简介等）
  - 修改密码
  - 绑定/解绑第三方账号
  - 隐私设置

### 2. 我的作品
- **功能描述：** 管理用户的作品
- **功能包括：**
  - 我的文档列表
  - 我的视频列表
  - 我的工具列表
  - 作品编辑/删除
  - 作品状态管理

### 3. 收藏与关注
- **功能描述：** 管理用户的收藏和关注
- **功能包括：**
  - 收藏的文档
  - 收藏的视频
  - 收藏的工具
  - 关注的用户
  - 粉丝列表

### 4. 消息通知
- **功能描述：** 查看用户的消息通知
- **功能包括：**
  - 系统消息
  - 评论通知
  - 点赞通知
  - 关注通知
  - 消息已读/未读

### 5. 设置
- **功能描述：** 系统设置
- **功能包括：**
  - 主题设置（深色/浅色）
  - 通知设置
  - 语言设置
  - 账号安全
