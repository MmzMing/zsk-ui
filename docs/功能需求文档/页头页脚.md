# 页头页脚

## 一、页头

### 1.1 前台页头（BasicLayout）

前台页头位于 `BasicLayout` 组件中，高度为 60px，固定在顶部，具有以下功能：

#### 1.1.1 左侧区域

| 功能 | 说明 |
|------|------|
| 品牌Logo | 显示网站Logo，支持hover动画，暗色模式下自动反转颜色 |
| 动态标题 | 使用 WordRotate 组件循环显示"知识库小破站"和"欢迎您的到来" |

#### 1.1.2 中间导航区域

| 功能 | 说明 |
|------|------|
| 桌面端导航 | 显示首页、知识库、制作简历、AI、关于博主等导航项，支持图标+文字展示 |
| 移动端导航 | 只显示图标，响应式适配 |
| 导航高亮 | 当前页面导航项显示主题色并带有底部边框 |

#### 1.1.3 右侧操作区

| 功能 | 说明 |
|------|------|
| 主题切换 | 点击切换浅色/深色主题，支持圆形遮罩动画 |
| 语言切换 | 切换中文/English双语 |
| 主题色选择 | 下拉菜单选择5种预设主题色（默认蓝、清新绿、神秘紫、活力橙、少女粉），支持自定义颜色 |
| 用户菜单（已登录） | 头像下拉菜单，包含后台管理、个人中心、退出登录 |
| 登录按钮（未登录） | 跳转到登录页面 |

#### 1.1.4 交互特性

| 特性 | 说明 |
|------|------|
| 滚动隐藏 | 向下滚动时页头自动隐藏，向上滚动时显示 |
| 粘性定位 | 始终固定在顶部，z-index为30 |
| 平滑过渡 | 所有状态变化都有0.2秒的过渡动画 |

### 1.2 后台页头（AdminLayout）

后台页头位于 `AdminLayout` 组件中，高度为 56px，具有以下功能：

#### 1.2.1 左侧区域

| 功能 | 说明 |
|------|------|
| 移动端菜单 | 点击弹出侧边栏菜单（仅移动端显示） |
| 面包屑导航 | 显示当前页面层级路径（如：后台管理 > 数据概览 > 仪表盘） |

#### 1.2.2 中间导航区域

| 功能 | 说明 |
|------|------|
| 顶部导航（水平/混合布局） | 显示管理模块分类，hover显示子菜单下拉列表 |

#### 1.2.3 右侧操作区

| 功能 | 说明 |
|------|------|
| 全屏切换 | 切换全屏/退出全屏模式 |
| 消息中心 | 消息通知入口（开发中） |
| 系统设置 | 打开系统设置面板 |
| 用户菜单 | 头像下拉菜单，包含返回前台、个人中心、退出登录 |

## 五、技术要点总结

1. **响应式设计**：完美适配桌面端和移动端
2. **动画效果**：使用 Framer Motion 实现流畅的过渡动画
3. **状态管理**：使用 Zustand 管理布局、主题等全局状态
4. **本地持久化**：所有用户配置自动保存到 localStorage
5. **主题适配**：所有组件支持浅色/深色双主题
6. **灵活配置**：多种布局模式、配色方案可选
