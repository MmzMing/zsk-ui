# 后台-用户管理

**文件位置：** `src/pages/Admin/Personnel/User.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 用户列表
- **功能描述：** 展示系统用户列表，支持分页、搜索、筛选
- **功能包括：**
  - 分页展示用户列表
  - 按用户名、手机号、邮箱搜索
  - 按用户状态、角色筛选
  - 按创建时间、登录时间排序

### 2. 用户管理
- **功能描述：** 管理系统用户
- **功能包括：**
  - 新增用户
  - 编辑用户信息
  - 删除用户
  - 批量删除
  - 重置用户密码
  - 启用/禁用用户

### 3. 用户详情
- **功能描述：** 查看和编辑用户详细信息
- **展示内容：**
  - 用户基本信息
  - 用户角色
  - 用户权限
  - 登录日志
  - 操作日志

### 4. 角色分配
- **功能描述：** 为用户分配角色
- **功能包括：**
  - 查看用户角色
  - 分配角色
  - 移除角色
  - 批量分配

### 5. 导入导出
- **功能描述：** 用户数据导入导出
- **功能包括：**
  - 导出用户列表
  - 导入用户数据
  - 下载导入模板
