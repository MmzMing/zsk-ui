# 后台-文档上传

**文件位置：** `src/pages/Admin/Document/Upload.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 文档上传
- **功能描述：** 提供文档上传功能
- **上传方式：**
  - 拖拽上传
  - 点击选择文件
  - 支持多种文件格式

### 2. 文件格式
- **功能描述：** 支持多种文档格式
- **支持格式：**
  - PDF
  - Word (doc, docx)
  - Excel (xls, xlsx)
  - PowerPoint (ppt, pptx)
  - TXT
  - Markdown
  - 图片格式 (jpg, png, gif 等)

### 3. 上传进度
- **功能描述：** 显示上传进度
- **进度显示：**
  - 文件上传进度条
  - 上传速度
  - 剩余时间
  - 已上传大小

### 4. 批量上传
- **功能描述：** 支持批量上传文档
- **批量功能：**
  - 多文件同时上传
  - 上传队列管理
  - 暂停/继续上传
  - 取消上传

### 5. 上传设置
- **功能描述：** 文档上传设置
- **设置内容：**
  - 文档分类
  - 文档标签
  - 文档权限
  - 是否自动发布
