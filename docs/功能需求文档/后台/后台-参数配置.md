# 后台-参数配置

**文件位置：** `src/pages/Admin/System/Param.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 参数列表
- **功能描述：** 展示系统参数列表，支持分页、搜索、筛选
- **展示内容：**
  - 参数名称
  - 参数键名
  - 参数键值
  - 系统内置
  - 备注
  - 创建时间

### 2. 参数管理
- **功能描述：** 管理系统参数
- **功能包括：**
  - 新增参数
  - 编辑参数
  - 删除参数
  - 批量删除
  - 刷新参数缓存

### 3. 参数详情
- **功能描述：** 查看和编辑参数详细信息
- **编辑内容：**
  - 参数名称
  - 参数键名
  - 参数键值
  - 系统内置
  - 备注

### 4. 参数查询
- **功能描述：** 支持多条件查询参数
- **查询条件：**
  - 参数名称
  - 参数键名
  - 系统内置
