# 后台-视频上传

**文件位置：** `src/pages/Admin/Video/Upload.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 视频上传
- **功能描述：** 提供视频上传功能
- **上传方式：**
  - 拖拽上传
  - 点击选择文件
  - 支持多种视频格式

### 2. 视频格式
- **功能描述：** 支持多种视频格式
- **支持格式：**
  - MP4
  - AVI
  - MOV
  - MKV
  - FLV
  - WMV

### 3. 上传进度
- **功能描述：** 显示上传进度
- **进度显示：**
  - 文件上传进度条
  - 上传速度
  - 剩余时间
  - 已上传大小

### 4. 视频处理
- **功能描述：** 视频上传后处理
- **处理功能：**
  - 视频转码
  - 生成封面
  - 生成缩略图
  - 视频切片

### 5. 批量上传
- **功能描述：** 支持批量上传视频
- **批量功能：**
  - 多文件同时上传
  - 上传队列管理
  - 暂停/继续上传
  - 取消上传

### 6. 上传设置
- **功能描述：** 视频上传设置
- **设置内容：**
  - 视频分类
  - 视频标签
  - 视频权限
  - 是否自动发布
