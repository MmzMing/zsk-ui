# 后台-QQ机器人

**文件位置：** `src/pages/Admin/Bot/QQBot.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 机器人配置
- **功能描述：** 配置 QQ 机器人
- **配置内容：**
  - 机器人 QQ 号
  - 机器人名称
  - 机器人头像
  - 机器人描述
  - 机器人状态

### 2. 机器人管理
- **功能描述：** 管理 QQ 机器人
- **管理功能：**
  - 启动机器人
  - 停止机器人
  - 重启机器人
  - 查看机器人状态
  - 重新登录

### 3. 消息管理
- **功能描述：** 管理机器人消息
- **消息功能：**
  - 查看消息记录
  - 发送消息
  - 接收消息
  - 消息统计
  - 消息搜索

### 4. 自动回复
- **功能描述：** 配置自动回复规则
- **回复功能：**
  - 关键词回复
  - 欢迎语设置
  - 定时消息
  - 群消息管理
  - 私聊消息管理

### 5. 群组管理
- **功能描述：** 管理 QQ 群组
- **群组功能：**
  - 查看群组列表
  - 群组信息管理
  - 群成员管理
  - 群消息管理
  - 群公告管理

### 6. 好友管理
- **功能描述：** 管理 QQ 好友
- **好友功能：**
  - 查看好友列表
  - 好友信息管理
  - 添加好友
  - 删除好友
  - 好友消息管理
