# 后台-令牌管理

**文件位置：** `src/pages/Admin/System/Token.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 令牌列表
- **功能描述：** 展示系统令牌列表，支持分页、搜索、筛选
- **展示内容：**
  - 用户信息
  - 令牌类型
  - 令牌状态
  - 创建时间
  - 过期时间
  - 最后使用时间

### 2. 令牌查询
- **功能描述：** 支持多条件查询令牌
- **查询条件：**
  - 用户名称
  - 令牌类型
  - 令牌状态
  - 创建时间范围
  - 过期时间范围

### 3. 令牌管理
- **功能描述：** 管理系统令牌
- **功能包括：**
  - 查看令牌详情
  - 撤销令牌
  - 批量撤销令牌
  - 刷新令牌列表

### 4. 令牌详情
- **功能描述：** 查看令牌详细信息
- **详情内容：**
  - 令牌基本信息
  - 用户信息
  - 令牌权限
  - 创建时间
  - 过期时间
  - 使用记录
