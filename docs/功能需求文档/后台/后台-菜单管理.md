# 后台-菜单管理

**文件位置：** `src/pages/Admin/Personnel/Menu.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 菜单树展示
- **功能描述：** 以树形结构展示系统菜单，支持展开/折叠
- **展示内容：**
  - 菜单层级关系
  - 菜单名称
  - 菜单图标
  - 菜单排序
  - 菜单状态

### 2. 菜单管理
- **功能描述：** 管理系统菜单
- **功能包括：**
  - 新增一级菜单（直接调用后端接口）
  - 新增子菜单
  - 编辑菜单信息
  - 删除菜单
  - 拖拽排序菜单

### 3. 菜单编辑
- **功能描述：** 编辑菜单详细信息
- **编辑内容：**
  - 菜单名称
  - 菜单图标
  - 路由地址
  - 组件路径
  - 权限标识
  - 排序号
  - 菜单状态
  - 是否缓存
  - 是否外链
  - 显示状态

### 4. 配置保存
- **功能描述：** 统一保存菜单配置
- **功能包括：**
  - 本地状态管理（编辑时不立即调用后端）
  - 保存配置时批量提交（调用后端 PUT 方法）
  - 取消编辑时恢复原始数据
  - 删除 Mock 数据依赖

### 5. 图标选择
- **功能描述：** 提供图标选择功能
- **功能包括：**
  - 内置图标库
  - 图标搜索
  - 图标预览
  - 自定义图标上传
