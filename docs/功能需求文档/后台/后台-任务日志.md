# 后台-任务日志

**文件位置：** `src/pages/Admin/Monitor/JobLog.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 任务日志列表
- **功能描述：** 展示定时任务执行日志列表，支持分页、搜索、筛选
- **展示内容：**
  - 任务名称
  - 任务组
  - 调用目标
  - 执行状态
  - 执行时间
  - 执行时长
  - 异常信息

### 2. 日志查询
- **功能描述：** 支持多条件查询任务日志
- **查询条件：**
  - 任务名称
  - 任务组
  - 执行状态
  - 执行时间范围

### 3. 日志详情
- **功能描述：** 查看任务执行详情
- **详情内容：**
  - 任务基本信息
  - 执行参数
  - 执行结果
  - 异常堆栈信息
  - 执行日志

### 4. 日志管理
- **功能描述：** 管理任务日志
- **功能包括：**
  - 删除日志
  - 批量删除
  - 清空日志
  - 导出日志
