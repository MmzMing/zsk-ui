# 后台-文档审核

**文件位置：** `src/pages/Admin/Document/Review.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 待审核文档列表
- **功能描述：** 展示待审核文档列表，支持分页、搜索、筛选
- **展示内容：**
  - 文档标题
  - 文档封面
  - 作者信息
  - 文档分类
  - 提交时间
  - 审核状态

### 2. 文档查询
- **功能描述：** 支持多条件查询待审核文档
- **查询条件：**
  - 文档标题
  - 作者
  - 文档分类
  - 审核状态
  - 提交时间范围

### 3. 文档审核
- **功能描述：** 审核文档
- **审核操作：**
  - 通过审核
  - 拒绝审核
  - 查看文档详情
  - 填写审核意见

### 4. 审核详情
- **功能描述：** 查看文档审核详情
- **详情内容：**
  - 文档基本信息
  - 文档内容
  - 提交时间
  - 审核历史
  - 审核意见
