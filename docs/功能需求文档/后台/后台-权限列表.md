# 后台-权限列表

**文件位置：** `src/pages/Admin/System/Permission.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 权限树展示
- **功能描述：** 以树形结构展示系统权限，支持展开/折叠
- **展示内容：**
  - 权限层级关系
  - 权限名称
  - 权限标识
  - 权限类型
  - 权限状态

### 2. 权限管理
- **功能描述：** 管理系统权限
- **功能包括：**
  - 新增权限
  - 编辑权限
  - 删除权限
  - 启用/禁用权限

### 3. 权限编辑
- **功能描述：** 编辑权限详细信息
- **编辑内容：**
  - 权限名称
  - 权限标识
  - 上级权限
  - 权限类型（目录、菜单、按钮）
  - 路由地址
  - 组件路径
  - 权限状态
  - 备注

### 4. 权限查询
- **功能描述：** 支持多条件查询权限
- **查询条件：**
  - 权限名称
  - 权限标识
  - 权限类型
  - 权限状态
