# 后台-视频列表

**文件位置：** `src/pages/Admin/Video/List.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 视频列表
- **功能描述：** 展示视频列表，支持分页、搜索、筛选
- **展示内容：**
  - 视频标题
  - 视频封面
  - 作者信息
  - 视频分类
  - 视频状态
  - 发布时间
  - 观看次数
  - 点赞数

### 2. 视频查询
- **功能描述：** 支持多条件查询视频
- **查询条件：**
  - 视频标题
  - 作者
  - 视频分类
  - 视频状态
  - 发布时间范围

### 3. 视频管理
- **功能描述：** 管理视频
- **功能包括：**
  - 新增视频
  - 编辑视频
  - 删除视频
  - 批量删除
  - 上架/下架视频
  - 推荐视频
  - 置顶视频

### 4. 视频详情
- **功能描述：** 查看视频详情
- **详情内容：**
  - 视频基本信息
  - 视频播放
  - 视频分类
  - 视频标签
  - 视频评论
