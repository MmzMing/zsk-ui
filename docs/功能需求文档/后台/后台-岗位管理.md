# 后台-岗位管理

**文件位置：** `src/pages/Admin/Personnel/Post.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 岗位列表
- **功能描述：** 展示系统岗位列表，支持分页、搜索、筛选
- **功能包括：**
  - 分页展示岗位列表
  - 按岗位名称、岗位编码搜索
  - 按岗位状态筛选
  - 按创建时间排序

### 2. 岗位管理
- **功能描述：** 管理系统岗位
- **功能包括：**
  - 新增岗位
  - 编辑岗位信息
  - 删除岗位
  - 启用/禁用岗位

### 3. 岗位编辑
- **功能描述：** 编辑岗位详细信息
- **编辑内容：**
  - 岗位名称
  - 岗位编码
  - 岗位排序
  - 岗位状态
  - 岗位备注

### 4. 岗位用户
- **功能描述：** 管理岗位下的用户
- **功能包括：**
  - 查看岗位用户列表
  - 岗位用户统计
