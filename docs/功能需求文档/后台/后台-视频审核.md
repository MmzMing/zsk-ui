# 后台-视频审核

**文件位置：** `src/pages/Admin/Video/Review.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 待审核视频列表
- **功能描述：** 展示待审核视频列表，支持分页、搜索、筛选
- **展示内容：**
  - 视频标题
  - 视频封面
  - 作者信息
  - 视频分类
  - 提交时间
  - 审核状态

### 2. 视频查询
- **功能描述：** 支持多条件查询待审核视频
- **查询条件：**
  - 视频标题
  - 作者
  - 视频分类
  - 审核状态
  - 提交时间范围

### 3. 视频审核
- **功能描述：** 审核视频
- **审核操作：**
  - 通过审核
  - 拒绝审核
  - 查看视频详情
  - 播放视频
  - 填写审核意见

### 4. 审核详情
- **功能描述：** 查看视频审核详情
- **详情内容：**
  - 视频基本信息
  - 视频播放
  - 提交时间
  - 审核历史
  - 审核意见
