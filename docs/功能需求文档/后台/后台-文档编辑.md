# 后台-文档编辑

**文件位置：** `src/pages/Admin/Document/Edit.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 文档编辑
- **功能描述：** 提供文档编辑功能
- **编辑内容：**
  - 文档标题
  - 文档封面
  - 文档分类
  - 文档标签
  - 文档摘要
  - 文档内容

### 2. 富文本编辑
- **功能描述：** 提供富文本编辑功能
- **编辑器功能：**
  - 文本格式化（加粗、斜体、下划线等）
  - 标题层级
  - 列表（有序、无序）
  - 图片上传
  - 视频上传
  - 链接插入
  - 代码块
  - 表格
  - 引用

### 3. 文档预览
- **功能描述：** 提供文档预览功能
- **预览功能：**
  - 实时预览
  - 全屏预览
  - 移动端预览

### 4. 文档保存
- **功能描述：** 保存文档
- **保存功能：**
  - 自动保存
  - 手动保存
  - 保存草稿
  - 发布文档

### 5. 文档设置
- **功能描述：** 文档设置
- **设置内容：**
  - 文档状态（草稿、已发布）
  - 是否推荐
  - 是否置顶
  - 阅读权限
  - 评论权限
