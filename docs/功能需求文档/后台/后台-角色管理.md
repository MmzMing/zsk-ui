# 后台-角色管理

**文件位置：** `src/pages/Admin/Personnel/Role.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 角色列表
- **功能描述：** 展示系统角色列表，支持分页、搜索、筛选
- **功能包括：**
  - 分页展示角色列表
  - 按角色名称、角色标识搜索
  - 按角色状态筛选
  - 按创建时间排序

### 2. 角色管理
- **功能描述：** 管理系统角色
- **功能包括：**
  - 新增角色
  - 编辑角色信息
  - 删除角色
  - 启用/禁用角色
  - 角色数据权限配置

### 3. 角色权限
- **功能描述：** 配置角色权限
- **功能包括：**
  - 查看角色权限
  - 分配菜单权限
  - 分配按钮权限
  - 权限树形展示
  - 批量权限分配

### 4. 角色用户
- **功能描述：** 管理角色下的用户
- **功能包括：**
  - 查看角色用户列表
  - 添加用户到角色
  - 从角色移除用户
  - 批量添加/移除
