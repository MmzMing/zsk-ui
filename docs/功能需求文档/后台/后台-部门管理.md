# 后台-部门管理

**文件位置：** `src/pages/Admin/Personnel/Dept.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 部门树展示
- **功能描述：** 以树形结构展示组织部门，支持展开/折叠
- **展示内容：**
  - 部门层级关系
  - 部门名称
  - 部门负责人
  - 部门状态
  - 部门排序

### 2. 部门管理
- **功能描述：** 管理组织部门
- **功能包括：**
  - 新增部门
  - 编辑部门信息
  - 删除部门
  - 启用/禁用部门
  - 拖拽排序部门

### 3. 部门编辑
- **功能描述：** 编辑部门详细信息
- **编辑内容：**
  - 部门名称
  - 上级部门
  - 部门负责人
  - 联系电话
  - 邮箱
  - 排序号
  - 部门状态

### 4. 部门用户
- **功能描述：** 管理部门下的用户
- **功能包括：**
  - 查看部门用户列表
  - 查看下级部门用户
  - 部门用户统计
