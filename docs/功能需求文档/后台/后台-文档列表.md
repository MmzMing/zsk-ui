# 后台-文档列表

**文件位置：** `src/pages/Admin/Document/List.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 文档列表
- **功能描述：** 展示文档列表，支持分页、搜索、筛选
- **展示内容：**
  - 文档标题
  - 文档封面
  - 作者信息
  - 文档分类
  - 文档状态
  - 发布时间
  - 阅读量
  - 点赞数

### 2. 文档查询
- **功能描述：** 支持多条件查询文档
- **查询条件：**
  - 文档标题
  - 作者
  - 文档分类
  - 文档状态
  - 发布时间范围

### 3. 文档管理
- **功能描述：** 管理文档
- **功能包括：**
  - 新增文档
  - 编辑文档
  - 删除文档
  - 批量删除
  - 上架/下架文档
  - 推荐文档
  - 置顶文档

### 4. 文档详情
- **功能描述：** 查看文档详情
- **详情内容：**
  - 文档基本信息
  - 文档内容
  - 文档分类
  - 文档标签
  - 文档评论
