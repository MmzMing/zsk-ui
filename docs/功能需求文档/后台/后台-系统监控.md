# 后台-系统监控

**文件位置：** `src/pages/Admin/Ops/SystemMonitor.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 系统概览
- **功能描述：** 展示系统整体运行状态
- **展示内容：**
  - 系统运行时间
  - 在线用户数
  - 今日访问量
  - 系统负载
  - 健康状态

### 2. 性能监控
- **功能描述：** 监控系统性能指标
- **监控内容：**
  - CPU 使用率
  - 内存使用率
  - 磁盘使用率
  - 网络流量
  - 请求响应时间

### 3. 实时监控
- **功能描述：** 实时展示系统监控数据
- **监控方式：**
  - 实时数据刷新
  - 趋势图表展示
  - 告警阈值设置
  - 异常告警通知

### 4. 历史数据
- **功能描述：** 查看历史监控数据
- **数据展示：**
  - 时间范围选择
  - 历史数据图表
  - 数据导出
  - 数据对比分析
