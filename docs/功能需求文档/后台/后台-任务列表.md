# 后台-任务列表

**文件位置：** `src/pages/Admin/Monitor/Job.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 定时任务列表
- **功能描述：** 展示系统定时任务列表，支持分页、搜索
- **展示内容：**
  - 任务名称
  - 任务组
  - 调用目标
  - Cron 表达式
  - 任务状态
  - 下次执行时间
  - 上一次执行结果

### 2. 定时任务管理
- **功能描述：** 管理定时任务
- **功能包括：**
  - 新增任务
  - 编辑任务
  - 删除任务
  - 批量删除
  - 启用/禁用任务
  - 立即执行任务

### 3. 任务详情
- **功能描述：** 查看和编辑任务详细信息
- **编辑内容：**
  - 任务名称
  - 任务组
  - 调用目标
  - 方法参数
  - Cron 表达式
  - 任务状态
  - 备注信息

### 4. 任务执行
- **功能描述：** 任务执行相关功能
- **功能包括：**
  - 手动执行任务
  - 查看执行日志
  - 执行结果统计
