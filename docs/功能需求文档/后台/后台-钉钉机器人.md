# 后台-钉钉机器人

**文件位置：** `src/pages/Admin/Bot/DingTalkBot.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 机器人配置
- **功能描述：** 配置钉钉机器人
- **配置内容：**
  - Webhook 地址
  - 机器人名称
  - 机器人头像
  - 机器人描述
  - 加签密钥
  - 机器人状态

### 2. 机器人管理
- **功能描述：** 管理钉钉机器人
- **管理功能：**
  - 启用机器人
  - 禁用机器人
  - 测试连接
  - 查看机器人状态

### 3. 消息管理
- **功能描述：** 管理机器人消息
- **消息功能：**
  - 发送文本消息
  - 发送 Markdown 消息
  - 发送链接消息
  - 发送 ActionCard 消息
  - 发送 FeedCard 消息
  - 查看消息记录
  - 消息统计

### 4. 自动通知
- **功能描述：** 配置自动通知规则
- **通知功能：**
  - 系统告警通知
  - 业务事件通知
  - 定时通知
  - 通知模板管理
  - 通知历史记录

### 5. 群组管理
- **功能描述：** 管理钉钉群组
- **群组功能：**
  - 查看群组列表
  - 群组信息管理
  - 群成员管理
  - 群消息管理
