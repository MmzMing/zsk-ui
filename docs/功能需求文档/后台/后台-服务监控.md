# 后台-服务监控

**文件位置：** `src/pages/Admin/Monitor/Server.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 服务器信息
- **功能描述：** 展示服务器基本信息
- **展示内容：**
  - 服务器名称
  - 操作系统
  - 系统架构
  - 服务器 IP
  - 主机名

### 2. CPU 监控
- **功能描述：** 监控 CPU 使用情况
- **展示内容：**
  - CPU 型号
  - CPU 核心数
  - CPU 使用率
  - CPU 使用率趋势图

### 3. 内存监控
- **功能描述：** 监控内存使用情况
- **展示内容：**
  - 总内存
  - 已用内存
  - 可用内存
  - 内存使用率
  - 内存使用率趋势图

### 4. 磁盘监控
- **功能描述：** 监控磁盘使用情况
- **展示内容：**
  - 磁盘总容量
  - 已用容量
  - 可用容量
  - 磁盘使用率
  - 磁盘分区信息

### 5. 系统监控
- **功能描述：** 监控系统运行状态
- **展示内容：**
  - 系统运行时间
  - 进程数
  - 线程数
  - 系统负载
  - 网络状态
