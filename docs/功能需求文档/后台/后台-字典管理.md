# 后台-字典管理

**文件位置：** `src/pages/Admin/System/Dict.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 字典列表
- **功能描述：** 展示系统字典列表，支持分页、搜索
- **展示内容：**
  - 字典名称
  - 字典编码
  - 字典状态
  - 备注
  - 创建时间

### 2. 字典管理
- **功能描述：** 管理系统字典
- **功能包括：**
  - 新增字典
  - 编辑字典
  - 删除字典
  - 批量删除
  - 启用/禁用字典

### 3. 字典数据
- **功能描述：** 管理字典数据项
- **功能包括：**
  - 查看字典数据列表
  - 新增字典数据
  - 编辑字典数据
  - 删除字典数据
  - 字典数据排序

### 4. 字典详情
- **功能描述：** 查看和编辑字典详细信息
- **编辑内容：**
  - 字典名称
  - 字典编码
  - 字典状态
  - 备注

### 5. 字典数据详情
- **功能描述：** 查看和编辑字典数据详细信息
- **编辑内容：**
  - 字典标签
  - 字典键值
  - 字典排序
  - 样式属性
  - 备注
