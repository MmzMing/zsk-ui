# 后台-操作日志

**文件位置：** `src/pages/Admin/Monitor/Operlog.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 操作日志列表
- **功能描述：** 展示系统操作日志列表，支持分页、搜索、筛选
- **展示内容：**
  - 操作人
  - 操作模块
  - 操作类型
  - 操作状态
  - 操作时间
  - 操作 IP
  - 操作地点

### 2. 日志查询
- **功能描述：** 支持多条件查询操作日志
- **查询条件：**
  - 操作人
  - 操作模块
  - 操作类型
  - 操作状态
  - 操作时间范围

### 3. 日志详情
- **功能描述：** 查看操作详细信息
- **详情内容：**
  - 操作基本信息
  - 请求参数
  - 响应结果
  - 请求方式
  - 请求 URL
  - 操作方法
  - 执行时长

### 4. 日志管理
- **功能描述：** 管理操作日志
- **功能包括：**
  - 删除日志
  - 批量删除
  - 清空日志
  - 导出日志
