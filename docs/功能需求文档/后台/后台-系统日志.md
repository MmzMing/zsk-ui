# 后台-系统日志

**文件位置：** `src/pages/Admin/Ops/SystemLog.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 系统日志列表
- **功能描述：** 展示系统日志列表，支持分页、搜索、筛选
- **展示内容：**
  - 日志级别
  - 日志类型
  - 日志内容
  - 日志时间
  - 模块名称
  - 操作人

### 2. 日志查询
- **功能描述：** 支持多条件查询系统日志
- **查询条件：**
  - 日志级别（DEBUG、INFO、WARN、ERROR）
  - 日志类型
  - 模块名称
  - 操作人
  - 日志时间范围

### 3. 日志详情
- **功能描述：** 查看系统日志详细信息
- **详情内容：**
  - 日志基本信息
  - 日志内容
  - 堆栈信息
  - 上下文信息
  - 请求信息

### 4. 日志管理
- **功能描述：** 管理系统日志
- **功能包括：**
  - 删除日志
  - 批量删除
  - 清空日志
  - 导出日志
  - 日志搜索
