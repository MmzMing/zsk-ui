# 后台-缓存列表

**文件位置：** `src/pages/Admin/Monitor/Cache.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 缓存信息
- **功能描述：** 展示系统缓存信息
- **展示内容：**
  - 缓存名称
  - 缓存键数量
  - 缓存命中率
  - 缓存大小

### 2. 缓存键列表
- **功能描述：** 展示缓存键列表，支持搜索
- **展示内容：**
  - 缓存键
  - 缓存值（摘要）
  - 过期时间
  - 剩余过期时间

### 3. 缓存管理
- **功能描述：** 管理系统缓存
- **功能包括：**
  - 查看缓存详情
  - 删除缓存键
  - 清空缓存
  - 批量删除缓存
  - 刷新缓存列表

### 4. 缓存详情
- **功能描述：** 查看缓存详细信息
- **详情内容：**
  - 缓存键
  - 缓存值（完整内容）
  - 缓存类型
  - 过期时间
  - 创建时间
