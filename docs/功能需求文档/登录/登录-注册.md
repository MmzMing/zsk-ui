# 登录-注册

**文件位置：** `src/pages/Auth/Register.tsx`  
**最后更新日期：** 2026-02-16

---

## 功能需求

### 1. 注册表单
- **功能描述：** 提供用户注册功能
- **注册方式：**
  - 用户名/密码注册
  - 手机号/验证码注册
  - 邮箱/验证码注册

### 2. 表单验证
- **功能描述：** 表单输入验证
- **验证内容：**
  - 用户名/手机号/邮箱格式验证
  - 密码强度验证
  - 确认密码一致性验证
  - 验证码格式验证
  - 必填项验证
  - 用户名唯一性验证
  - 手机号/邮箱唯一性验证

### 3. 用户协议
- **功能描述：** 用户协议和隐私政策
- **协议内容：**
  - 用户服务协议
  - 隐私政策
  - 必须同意才能注册

### 4. 验证码
- **功能描述：** 提供验证码功能
- **验证码类型：**
  - 图形验证码
  - 短信验证码
  - 邮箱验证码
  - 验证码倒计时

### 5. 交互设计
- **功能描述：** 优秀的注册交互体验
- **交互特性：**
  - 注册方式切换
  - 加载状态提示
  - 注册成功跳转
  - 注册失败提示
  - 登录入口
